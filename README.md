# 🚀 英語重組大挑戰：神秘人 Delta 的任務 🌌

歡迎來到**英語重組大挑戰**！這是一個結合太空冒險與英語學習的互動遊戲。不管你是想挑戰自我的小勇者，還是正在尋找趣味教學工具的老師，這裡都有最適合你的任務！

## 🎮 遊戲特色

* **🌌 兩大任務模式**：
* **星際挑戰賽**：挑戰由老師親自設計的官方題庫，看看誰才是最強星際特工！
* **自主訓練營**：你可以上傳自己的 Excel 或 Google Sheets 題庫，進行專屬訓練。


* **🌓 護眼雙色模式**：
* **大自然模式**：白天的森林綠，讓你學習不疲勞。
* **Cyberpunk 模式**：酷炫的夜間模式，讓你像電腦駭客一樣帥氣地解題。


* **🧩 超順手拖拽設計**：
* 方塊會乖乖排隊！拖動字詞時會自動顯示「空格提示」，再也不怕方塊疊在一起。


* **🎲 神秘人 Delta 的挑戰**：
* 這是終極關卡！系統會從簡單到困難隨機抽出 10 題，你有信心過關嗎？


* **📜 個人戰報系統**：
* 遊戲會幫你記住最近 10 次的成績，還會幫你找出最難、最花時間的題目，讓你變強更有感！



---

## 👩‍🏫 給老師的說明 (How to Play)

1. **開啟遊戲**：直接點擊我們提供的網址即可開始。
2. **語言選擇**：首頁可以切換「中文」或「English」介面。
3. **準備題庫**：如果您想讓學生玩特定的內容，請準備一個 Excel 檔，並確認：
* **A 欄**：數字（題號）。
* **B 欄**：正確英文句子（字與字之間請留空格）。
* **C 欄**：中文翻譯或提示。


4. **安全無憂**：本遊戲不收集個人資料，所有成績僅儲存在學生自己的電腦中。

---

## 🧒 給小朋友的挑戰指南

1. **輸入名字**：選好模式後，輸入一個帥氣的暱稱。
2. **動動手指**：用滑鼠或手指拖動單字，把它們放進正確的位置。
3. **看提示**：如果忘記怎麼拼，記得看下方的 **Person B** 提示！
4. **變身駭客**：覺得螢幕太亮？點擊右上角的 **「月亮按鈕」** 進入 Cyberpunk 模式！
